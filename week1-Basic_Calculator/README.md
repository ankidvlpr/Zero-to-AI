# 🎉 CLI Calculator - My First Program!

## About this Project

This is my first program! I'm excited to start my journey in programming, and this CLI calculator represents the beginning of my coding career. It's a simple yet functional command-line calculator built with Python that performs basic arithmetic operations.

## Overview

This calculator is a straightforward tool that takes user input and performs mathematical operations. It's designed with clean logic and proper error handling to create a smooth user experience.

## Features


# Supports basic arithmetic operations: **Add (+)**, **Subtract (-)**, **Multiply (*)**, **Divide (/)**
- Accepts both integers and decimal numbers
- User-friendly prompts and error messages

🔒 **Error Handling**
- Validates input format (ensures "number operator number" format)
- Handles invalid numbers gracefully
- Prevents division by zero errors
- Provides clear error messages for better user guidance

🚪 **Exit Command** - Type 'exit' to quit the program anytime

## How to Use

1. Run the program:
   ```bash
   python calculator.py
   ```

2. Enter your calculation in the format: `number operator number`
   - Example: `2 + 3`
   - Example: `10 * 5`
   - Example: `20 / 4`

3. The program will display the result and ask for the next calculation

4. Type `exit` to close the program

## Supported Operations

| Operator | Operation | Example |
|----------|-----------|---------|
| `+` | Addition | `5 + 3 = 8` |
| `-` | Subtraction | `10 - 3 = 7` |
| `*` | Multiplication | `4 * 5 = 20` |
| `/` | Division | `20 / 4 = 5` |
