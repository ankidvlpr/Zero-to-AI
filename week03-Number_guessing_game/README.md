# Number Guessing Game 🎯

A simple command-line number guessing game built with Python.

The program generates a random number between 1 and 100, and the player must guess it within a limited number of attempts based on the selected difficulty level.

## Features

- Random number generation (1–100)
- Difficulty levels:
  - Easy → 10 attempts
  - Hard → 5 attempts
- Input validation
- Attempt tracking
- Score tracking
- High score system
- Replay option

## What I learned

This project helped me understand how to:

- Break a problem into small steps
- Use loops and conditions to control game flow
- Work with functions and variables
- Handle user input safely
- Track score and game state across rounds
- Think logically to build a complete working program
